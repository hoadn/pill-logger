Recent changes

3rd April 2014 - v1.2.3

  • If a size is an exact number, the .0 is no longer shown
  • ml is now shown throughout the app if selected as the units for medicine
  • Rotating with a dialog open no longer crashes the app
  • Reduced size of app by removing unused resources
  • Other bug fixes

25th March 2014 - v1.2

  • NEW: Statistics tab replaces the pretty much useless charts tab. Check it out and let us know what you think.
  • Some big performance increases, should hopefully make this a more snappy experience for everyone.

6th March 2014 - v1.1.0

  • Reminders! When recording usage you can now set a reminder.
  • Information dialog. Tapping a Consumption or Medicine will now show dialog with actions/information
  • Can now have black and white as medicine colours
  • Can select medicine colour when creating new medicine
  • Other small bug fixes
